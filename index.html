AEC Coding Club
Assam Engineering College is the first engineering institution of the North East India. With the advent of the IT boom, students started developing interests in the world of binary numbers and programming languages. As a push from this knack towards programming languages and softwares, the AEC Coding Club, was constituted in February 16'.
Initially with only a few members, now it is one of the most thriving club in the college. It aims to ease out the fear for programming people develop and to build a sustainable peer-to-peer help group among the students as well as the alumnis.
The club actively involves in conducting workshops on different programming languages like C, C++ and Python, and on latest on demand fields like web development, Android application development, Machine Learning, etc. Throughout the year, the club organises a few competitive events as well like CodeWar, which is the competitive programming competition of AEC Coding Club, held during Udbhavanam, the annual technical festival.

AEC CTF 2019
 AEC CTF (Capture The Flag) was a kind of online jeopardy style information security competition where contestants were asked to solve a variety of problems including – Scavenger Hunt, Basic Programming, Riddle Solving and Hacking to steal data. The contestant had to complete a series of challenges using the prior techniques and as the name suggest find a specific piece of text known as “flag”.
Induction Program 2019
CodeWar 2020
Coding Club was back with the much-awaited coding event under the banner of Udbhavanam 8.0, the Annual Technical fest of Assam Engineering College, the “CodeWar 2020”. It was a 2-round competitive programming event where anybody can showcase their programming knowledge, and/or experience of competitive programming and win exciting prizes.
Perplex 1.0
This time Coding Club is organizing a second event under Udbhavanam named Perplex 1.0. It is a puzzle solving challenge where thinking out of the box earns you rewards. The contestant doesn’t need to have any prior knowledge of Coding or Programming. Here all the coders and non-coders are welcomed.
Python ML Workshop
 Coding Club for the very first time organised a 2-day workshop on Machine Learning in collaboration with Google at Assam Engineering College. It was mentored by student mentors of our college from Google, Mr. Amartya Ranjan Saikia and Mr. Subhajit Saha
